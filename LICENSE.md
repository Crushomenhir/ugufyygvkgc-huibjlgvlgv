## › FAQ

**Q: Does it work with all games?**  
A: It works with most DirectX, OpenGL, and Vulkan applications. Some anti-cheat protected titles may require running the tool as administrator or testing compatibility.

**Q: Will this increase my FPS?**  
A: Frame generation can make motion appear smoother by inserting extra frames, but it does not increase the actual rendering performance of the application.

**Q: Is there input lag?**  
A: Minimal additional latency in most cases. Frame generation mode has very low overhead compared to traditional upscaling-only solutions.

**Q: Can I use it for video playback?**  
A: Yes. It works great with media players, browsers, and streaming applications to improve quality of lower-resolution videos.

**Q: Do I need a powerful GPU?**  
A: No. The tool is optimized to run efficiently. Basic scaling works on integrated graphics; advanced features perform better with a dedicated GPU.
