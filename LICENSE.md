## › FAQ

**Q: Does it work with exclusive fullscreen games?**  
A: No. The game must run in windowed or borderless windowed mode.

**Q: Will this get me banned in online games?**  
A: It only captures and modifies the output window. It doesn't inject into the game process. Use at your own risk and check the game's rules.

**Q: Can I use it for non-gaming applications?**  
A: Yes. Any windowed app can be scaled — videos, emulators, desktop apps.

**Q: Does it require administrator rights?**  
A: Usually not, but running as admin can help with capturing certain protected windows.

**Q: How do I turn it off quickly?**  
A: Use the global hotkey or click the tray icon to toggle scaling.
